# 快速启动指南

## ⚡ 5分钟快速开始

### 1. 安装依赖（约1分钟）

```bash
pip install PyQt5 httpx pydantic sqlalchemy PyMuPDF python-docx pynput pyperclip chardet
```

或使用requirements.txt：

```bash
pip install -r requirements.txt
```

### 2. 获取API密钥（约3分钟）

#### 百度翻译（推荐，免费配额充足）

1. 访问：https://fanyi-api.baidu.com/
2. 注册/登录 → 管理控制台 → 开发者信息
3. 创建应用，复制 **APP ID** 和 **密钥**

#### 有道翻译（可选）

1. 访问：https://ai.youdao.com/
2. 注册/登录 → 应用管理 → 创建应用
3. 复制 **APP Key** 和 **APP Secret**

### 3. 启动应用（约1分钟）

```bash
python main.py
```

首次启动后：

1. 点击菜单 **设置 → 偏好设置**
2. 填入百度翻译的 APP ID 和密钥
3. （可选）填入有道翻译密钥
4. 点击"确定"保存

### 4. 开始使用

#### 方式一：主窗口翻译

- 在"文本翻译"标签页输入文字
- 点击"翻译"按钮
- 查看结果

#### 方式二：框选翻译（最常用）

1. 在任意应用（浏览器、Word、PDF等）选中文本
2. 按 **Ctrl+Alt+T**
3. 悬浮窗自动显示翻译

#### 方式三：文件翻译

1. 切换到"文件翻译"标签
2. 选择 TXT/DOCX/PDF 文件
3. 点击"翻译文件"
4. 保存结果

## 🎯 常见使用场景

### 场景1：阅读英文文档

1. 打开PDF/Word文档
2. 框选不懂的段落
3. 按 Ctrl+Alt+T 查看翻译

### 场景2：翻译整篇文档

1. 主窗口 → 文件翻译
2. 上传文件
3. 等待翻译完成
4. 导出结果

### 场景3：学习单词

1. 选中单个英文单词
2. 按 Ctrl+Alt+T
3. 查看音标、释义、例句

## 🔧 配置优化

### 提升翻译质量

1. 设置 → 默认引擎选择"百度"（质量更好）
2. 或选择"有道"（单词详解更丰富）

### 修改快捷键

1. 设置 → 全局快捷键
2. 修改为你喜欢的组合（如 `<ctrl>+<shift>+t`）
3. 保存后重启应用

### 设置默认语言

1. 设置 → 默认源语言：自动检测
2. 设置 → 默认目标语言：中文

## 💡 使用技巧

### 技巧1：系统托盘使用

- 关闭主窗口后，应用在托盘继续运行
- 双击托盘图标可重新打开
- 框选翻译功能始终可用

### 技巧2：翻译历史管理

- 所有翻译自动保存
- 可随时在"翻译历史"查看
- 支持关键词搜索
- 可导出为CSV用Excel打开

### 技巧3：批量翻译

- 使用"文件翻译"功能
- 支持大文件自动分块
- 进度条实时显示

## ❓ 常见问题

### Q: 翻译失败，提示"密钥未配置"

**A**: 请在设置中填写正确的API密钥

### Q: 框选翻译无反应

**A**:
1. 确认应用正在运行（查看系统托盘）
2. 确认快捷键无冲突
3. 重新选中文本再按快捷键

### Q: PDF翻译结果乱码

**A**: 仅支持纯文本PDF，扫描版PDF请先用OCR转换

### Q: 翻译速度慢

**A**:
1. 检查网络连接
2. 尝试切换翻译引擎
3. 大文件需等待分块翻译完成
